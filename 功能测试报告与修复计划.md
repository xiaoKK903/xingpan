### 4.3 具体修复任务

#### 任务 1：前端 API 路径修复

**目标**：修复前端调用的 API 路径错误

**涉及文件**：
- `frontend/src/api/*.js` - 所有 API 调用文件

**修复内容**：
1. `/api/horoscope/daily` → `/api/horoscope/today`
2. `/api/checkin` → `/api/checkin/sign`
3. `/api/invite/info` → `/api/invite/code`
4. `/api/topic-challenge/current` → `/api/topic-challenge/active`
5. `/api/leaderboards` → `/api/leaderboards/configs`
6. `/api/synastry-analysis/calculate` → `/api/synastry-analysis/calculate-and-analyze`
7. `/api/transit/current` → `/api/transit/calculate`
8. `/api/star-resonance/config` → `/api/star-resonance/element-info`
9. `/api/boss-battle/active` → `/api/boss-battle/hall`

#### 任务 2：合盘 API 参数格式修复

**目标**：修复合盘计算的参数格式错误

**涉及文件**：
- `frontend/src/api/synastry.js`

**修复内容**：
```javascript
// 错误格式
{
  person1_birth_date: "...",
  person1_birth_time: "...",
  ...
}

// 正确格式
{
  person_a: {
    name: "...",
    birth_date: "...",
    birth_time: "...",
    latitude: ...,
    longitude: ...,
    house_system: "placidus"
  },
  person_b: {
    name: "...",
    birth_date: "...",
    birth_time: "...",
    latitude: ...,
    longitude: ...,
    house_system: "placidus"
  }
}
```

#### 任务 3：中文编码修复

**目标**：修复页面中文乱码问题

**涉及文件**：
- `frontend/index.html` - 添加正确的 charset 声明
- `frontend/vite.config.js` - 配置正确的编码

**修复内容**：
```html
<!-- index.html -->
<meta charset="UTF-8" />
```
